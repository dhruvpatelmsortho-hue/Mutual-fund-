# Mutual Funds Live Dashboard

Live dashboard: 21 Indian + 2 foreign funds vs Sensex / Nifty / Midcap / Smallcap.
Shows today's NAV, all-time-high NAV, 1Y & 2Y rolling returns, and vs-benchmark
comparison. Index cards show current level + ATH. Auto-updates every weekday.

## Deploy (see instructions given in chat)
1. Upload all files (keep the `.github/workflows/` folder).
2. Settings > Actions > General > Workflow permissions > Read and write > Save.
3. Actions tab > Update Mutual Funds Dashboard > Run workflow (once).
4. Settings > Pages > Source: main / root > Save.
5. Open https://YOUR_USERNAME.github.io/YOUR_REPO/

Repo must be Public for free GitHub Pages.
